Introduction: R and Jupyter

Welcome to R, on HCC/Binder, through Jupyter.

"R is a free software environment for statistical computing and graphics"
www.r-project.org

"The Jupyter Notebook is an open-source web application that allows you to create and share documents that contain live code, equations, visualizations and narrative text"
www.jupyter.org

Here, we'll be touching upon some of the fundamentals of programming in the R language. Topics to be covered include: functions, structures, packages, manipulation, graphics, saving and loading.

Jupyter Notebook
Segregated into a kernel (execute code) and dashboard/editor (interface), Jupyter permits mixing real-time coding with interactive rich-text ouputs in self-contained .ipynb(JSON) documents (local or remote).
Kernels include Julia, Python, R, MATLAB/Octave, etc.
Interact with input cells (code, markdown, etc) that exhibit an order and mode - insert, edit, run.

Graphics
For visualizations package ggplot2 is preferred over base functions (plot(), lines(), barplot()).
Mapping data, which can come from multiple sources (dataframes), to graphic attributes (x, y, color, fill, group) is done through aesthetics (aes).
Layers (geoms, stats, position, annotation), scales, facets, coordinate systems, themes can all then be added with +.

Example geoms: geom_point, geom_line, geom_bar, geom_boxplot, geom_polygon, geom_path.
